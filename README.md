# OrbitSCP

Cross-platform SFTP / SCP / FTP client inspired by WinSCP with modern UI, built using Electron.

![OrbitSCP Banner](build/icon-1.0.2.png)

## ✨ Features

- 🔐 SFTP, SCP, and FTP support
- 🖥️ Works on Windows, Linux, and macOS
- ⚡ Fast file browsing and transfer
- 📝 Auto-upload on save editor workflow
- 📂 Dual-pane file manager
- 🔑 SSH key authentication support
- 📄 PPK to OpenSSH key conversion
- 🧩 Built-in terminal using xterm.js
- 👀 File icons and modern UI
- 🔄 Real-time file watching with auto sync
- 📦 Electron-based desktop app
